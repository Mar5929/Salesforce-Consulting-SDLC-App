# Overview

- We're currently refining requirements for a Salesforce Consulting SDLC application. I created a master PRD with sub-modules — sub-pages that the master PRD references for more content-heavy sections — and I'm in the process of breaking that master PRD down into epics, features, and smaller units of requirements. The goal is for AI agents like Claude Code to be able to execute code development and testing by pointing them directly at a requirement, with no guesswork or research needed on their end.

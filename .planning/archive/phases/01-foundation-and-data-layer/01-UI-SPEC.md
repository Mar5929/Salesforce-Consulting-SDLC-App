## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, sidebar item height |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks, wizard step spacing |
| 3xl | 64px | Page-level vertical spacing |

Exceptions: Sidebar width fixed at 240px. Touch targets minimum 36px height for buttons and interactive elements.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Label | 13px | 600 (semibold) | 1.2 |
| Body | 14px | 400 (regular) | 1.5 |
| Heading | 18px | 600 (semibold) | 1.2 |
| Page Title | 24px | 600 (semibold) | 1.2 |

Font stack: `"Inter", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

Notes:
- Body text at 14px/400 is the default for all paragraph content, table cells, form inputs, and descriptions.
- Labels at 13px/600 are used for form labels, sidebar navigation items, badge text, and metadata.
- Headings at 18px/600 are used for card titles, section headings, and wizard step titles.
- Page titles at 24px/600 are used for top-level page headings only (one per page).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | hsl(0, 0%, 100%) / #FFFFFF | Page backgrounds, content surfaces |
| Secondary (30%) | hsl(0, 0%, 98%) / #FAFAFA | Cards, sidebar background, table headers, hover states |
| Accent (10%) | hsl(220, 70%, 50%) / #2563EB | See reserved-for list below |
| Destructive | hsl(0, 84%, 60%) / #EF4444 | Destructive actions only |

### Secondary Palette

| Role | Value | Usage |
|------|-------|-------|
| Border | hsl(0, 0%, 90%) / #E5E5E5 | Card borders, dividers, input borders |
| Muted text | hsl(0, 0%, 45%) / #737373 | Secondary text, timestamps, placeholder text |
| Text primary | hsl(0, 0%, 9%) / #171717 | Headings, body text, primary labels |
| Text secondary | hsl(0, 0%, 32%) / #525252 | Descriptions, helper text |

### Accent Reserved For

- Primary CTA buttons ("Create Project", "Save", "Invite Member")
- Active sidebar navigation item (text color + left border indicator)
- Focus rings on interactive elements (inputs, buttons, selects)
- Link text on hover
- Wizard active step indicator
- Badge for current user's role

Accent is NOT used for: borders, backgrounds of non-interactive elements, icons in default state, or decorative purposes.

## Layout Contract

### App Shell (D-01, D-04)

```
+--[ 240px Sidebar ]--+--[ Main Content Area ]------+
|                      |                              |
| [Project Switcher]   |  [Page Title — 24px/600]    |
| ────────────────     |                              |
| [Nav Items]          |  [Content]                   |
|   Projects           |                              |
|   Settings           |                              |
|   Team               |                              |
|                      |                              |
| ────────────────     |                              |
| [User Menu]          |                              |
+----------------------+------------------------------+
```

- Sidebar: 240px fixed width, secondary background (#FAFAFA), border-right (#E5E5E5)
- Main content: fluid width, white background, 32px horizontal padding, 24px top padding
- Sidebar navigation items: 13px/600, 36px height, 16px left padding, 8px vertical gap
- Active nav item: accent text color + 2px left border in accent
- Project switcher: DropdownMenu at top of sidebar, 16px padding from edges

### Project Creation Wizard (D-03)

- Centered in main content area, max-width 560px
- Step indicator: horizontal dots/numbers showing 3 steps
- Active step: accent fill. Completed: accent outline with checkmark. Upcoming: muted border
- Step content in Card component with 24px padding
- Navigation: "Back" (outline button) left-aligned, "Next"/"Create Project" (accent button) right-aligned
- Final step CTA: "Create Project"

### Loading States

- Page-level: Skeleton components matching layout structure (sidebar skeleton + content skeleton)
- Component-level: Skeleton matching content shape (e.g., card skeleton, table row skeleton)
- Button loading: Disabled state with spinner icon replacing text, same button dimensions

### Error Boundaries

- Page-level error: Centered message with retry button. Uses Card component.
- Section-level error: Inline error message with retry link within the section.

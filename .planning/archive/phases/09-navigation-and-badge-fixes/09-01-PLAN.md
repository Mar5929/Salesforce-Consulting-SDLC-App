Declare badge count variables before the `if (projectId)` block:

```typescript
let questionReviewCount: number | undefined

```

Replace the existing `if (projectId)` block with a version that runs all three queries in parallel using `Promise.all`. Per D-04, this follows the established pattern. Per D-05 and D-06, the status filters are `"ANSWERED"` and `"OPEN"`:

```typescript
if (projectId) {
  try {
    const [member, qCount, dCount] = await Promise.all([
      getCurrentMember(projectId),
      prisma.question.count({
        where: { projectId, status: "ANSWERED" },
      }),
      prisma.defect.count({
        where: { projectId, status: "OPEN" },
      }),
    ])
    currentMemberRole = member.role
    questionReviewCount = qCount
    openDefectCount = dCount
  } catch {
    redirect("/")
  }
}
```

Pass the new props to AppShell in the return JSX:

```typescript
<AppShell
  currentMemberRole={currentMemberRole}
  activeProjectId={projectId ?? undefined}
  questionReviewCount={questionReviewCount}
  openDefectCount={openDefectCount}
>
```
